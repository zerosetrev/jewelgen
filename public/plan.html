The Problem

Beading is stuck in the dark ages

A $300 billion jewellery market with a fabrication layer that hasn't changed in centuries. Every necklace is still made the same way: one bead, one hand, one aching back at a time.

T

Time Drain

A single beaded necklace takes 30-90 minutes by hand. A batch of 20 for a market stall? That's an entire weekend gone. Time that should be spent designing, selling, or living.

R

Repetitive Strain

Hand beading causes eye fatigue, neck strain, and repetitive stress injuries. Professional beaders regularly develop carpal tunnel. The craft literally hurts the maker.

S

Scaling Is Impossible

Want to sell 100 necklaces? Hire helpers, train them, quality-check everything, manage the stress. The only way to scale hand beading is more hands. That doesn't scale.

E

Errors Are Expensive

One wrong bead on row 47 means unthreading half the necklace. Counting errors, pattern mistakes, and colour mismatches waste materials and morale.

The Insight

Beads fall down. That's free.

Every existing solution tries to pick beads up and place them — robot arms, tweezers, vacuum nozzles. Expensive. Complex. Fragile. We asked a different question: what if the beads just fell into place?

The Robot Approach

  • Pick-and-place robot arm ($5,000+)
  • Vision system for bead identification
  • Complex calibration and maintenance
  • Slow: pick, move, place, return, repeat
  • Bulky, heavy, intimidating
  • Breaks when misaligned by 0.5mm

The Gravity Approach

  • Vertical bead tubes + solenoid gates ($24)
  • Beads pre-sorted in tubes — no vision needed
  • Zero calibration — gravity is constant
  • Fast: gate opens, bead drops, sub-second
  • Toaster-sized, portable, friendly
  • Near-indestructible mechanics
How It Works

Five steps. Thirty seconds.

From screen to finished necklace. No skill required. No bead touching. No mistakes.

1

Design

Open the JewelGen app on any device. Choose your base (silk, wire, leather, chain), pick beads from 7 categories, arrange them on the visual necklace designer. Add a clasp. See a real-time preview of exactly what you'll get.

jewelgen.app -> visual designer -> drag beads -> preview
2

Load

Snap bead cartridges into the machine's vertical slots. Each cartridge holds one bead type — pre-sorted, pre-labelled. Load a base spool (silk thread, jewellery wire, leather cord). Attach a clasp to the starter clip.

cartridge slot 1: Lapis Lazuli 10mm (qty: 50)
cartridge slot 2: Gold Spacer 6mm (qty: 100)
cartridge slot 3: Pearl White 8mm (qty: 40)
base spool: Silk Thread — Cream
clasp: Gold Lobster
3

Send

Tap "Print" in the app. The design transfers wirelessly to the machine via Bluetooth. The machine validates: do you have enough beads? Is the right base loaded? Green light = ready.

app -> bluetooth -> ESP32 -> validate inventory -> ready
4

Print

The machine runs the design. For each bead in the sequence, the correct solenoid gate opens. One bead drops onto the horizontal string. Gate closes. String advances. Next bead. 20 beads in under 30 seconds. The clasp crimps automatically at both ends.

GATE_OPEN: slot_1 (lapis) -> BEAD_DROP -> STRING_ADVANCE: 12mm
GATE_OPEN: slot_2 (gold) -> BEAD_DROP -> STRING_ADVANCE: 8mm
GATE_OPEN: slot_3 (pearl) -> BEAD_DROP -> STRING_ADVANCE: 10mm
... (x20 beads) ...
CRIMP_END -> CUT -> RELEASE
5

Collect

The finished necklace drops into the padded output tray. Ready to wear, gift, or sell. Tap "Print Again" for another copy, or load a new design.
